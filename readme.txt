Creo una aplicación para el calculo de áreas. En este caso para triangulo, rectángulo y circulo.
Utilizamos como input un formulario. 
Dentro del formulario tenemos una lista, cada vez que seleccionamos un objeto este parece en la parte inferior.
Para los objetos que aparecen hemos cargado 3 svg con un circulo, cuadrado y rectángulo que permanecen ocultos hasta que la página detecta que existe un cambio en el formulario. En ese momento hace visible el sbg correspondiente.

Áreas:
    triangulo ==> b*h/2
    Rectángulo ==> b * h 
    circulo ==> π  * r2
